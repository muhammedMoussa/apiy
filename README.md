# Apiy 🚀
> a lightweight plugin that makes asynchronous api calls, one second of code!.

# Installation ⚙️

`npm install apiy` ||
`yarn apiy`

Then...

```
import { Apiy } from 'apiy';

const apiy = new Apiy();

api.get({ url: 'https://jsonplaceholder.typicode.com/todos/1' })
.then(res => console.log(res))
.catch(err => console.log(err))
...
```
